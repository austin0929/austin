JavaScript 陣列處理方法forEach, map, filter

Created2023-02-25|Updated2023-02-27
|Post View:

以下介紹這三種陣列使用方法

forEach

單純把陣列中的物件全部取出，無法return回傳值

map

map跟forEach很像，差別在map會把生成一個新陣列把物件帶入，需使用return 回傳值，不回傳是undefined

filter

可以篩選出你需要的物件並把所有判斷後true的物件組合成新的陣列 並回傳，

結論

由於map跟，filter陣列處理方式比較少使用，
做幾個範例並記錄，後面還有幾個陣列處理方式再慢慢補上。
