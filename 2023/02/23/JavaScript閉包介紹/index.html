JavaScript 閉包是什麼?

閉包簡單來說就是在函式中回傳一個函式，
透過封裝元素讓函式內記憶體不會被釋放 並可以不斷調用。

範例

透過封裝count這個變數，讓count可以不斷被計算 而且可以獨立出來，簡單來說david會有一個count變數，john也有一個count變數， 各自計算是不會互相干擾的，只能在myMoney函式內取用 ，雖然取用同一個函式，但都各自執行。
