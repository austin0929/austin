Vue Cli中使用VeeValidate套件

最近在接觸做表單練習，會需要驗證表單， 讓我們來學習一下如何使用VeeValidate套件吧

安裝VeeValidate

使用終端機或vsCode終端機內輸入:
npm install vee-validate

這邊的vee-validate並不包含驗證規則，所以我們要另外安裝 驗證規則套件，輸入以下指令
npm install @vee-validate/rules

接著把多國語系安裝
npm install @vee-validate/i18n

匯入VeeValidate

接下來要匯入VeeValidate到我們的Vue Cli中，這邊我們使用全域匯入， 區域匯入的話基本上大同小異，接著看匯入方式。
