在Vue Cli中加入Bootstrap Modal

以前使用Bootstrap都用HTML操控比較單純，改成Vue後變成要改成用JS來操作 ，步驟會多一點。

流程一

新增Modal元件

首先我們先去Bootstrap載入一個Modal的範例，並使用 ref 屬性加入到Modal上，以便待會取得Modal的DOM元素。

流程二

匯入Modal

從node_modules內的bootstrap/js/dist/modal匯入Modal
匯入路徑:
import Modal from 'bootstrap/js/dist/modal'

Modal實體化

接著撰寫JS，並使用mounted把modal實體化，然後使用 $refs 屬性 來指向模板內 ref="modal" 內的DOM，並使用this.modal傳入 自己data內的modal。

流程三

把Modal元件匯入到產品頁面

把剛剛的modal元件匯入到產品頁面上
匯入路徑:
import ProductModal from '../components/ProductModal.vue'

進行區域註冊

匯入後進行區域註冊

之後再模板內加入你進行註冊的標籤就可以渲染在畫面上了。

流程四

加入Modal開啟關閉方法

有畫面之後還需要按鈕來開啟關閉，回到Modal元件來加入methods方法。

完成後回到產品頁面，接著來觸發這些事件。

流程五

此標籤是在流程三中產品頁面進行區域註冊並渲染的標籤，現在要對他加入 ref 屬性， 之後會對他做操作。

用v-on觸發Modal事件

在模板內新增一個button標籤，接著使用 $refs 屬性 來取得DOM路徑，然後使用v-on:click來觸發showModal事件開啟。

這樣Modal的基本範例就完成了， 範例的路徑需要依照專案的不同來對應。
