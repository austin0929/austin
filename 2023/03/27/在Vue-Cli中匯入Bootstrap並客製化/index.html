|Post View:

Vue Cli中匯入Bootstrap

前言:
在vue中匯入bootstrap跟一般匯入方式有點不一樣，讓我們來試著做做看。

流程一安裝Bootstrap

先到官網找npm install bootstrap打開VsCode終端機 並安裝上去，記得先退出Vue Cli環境。

流程二匯入Bootstrap

完成後到app.vue把scss內的預設值清空後匯入
~bootstrap/scss/bootstrap
我們把原本的../node_modules/替換成~符號。

之後去bootstrap官網找範例嘗試看看是否有正確載入。

流程三客製化Bootstrap

1. assets資料夾下新增helpers資料夾，helpers資料夾內新增_variables.scss檔案。
2. assets資料夾下新增all.scss檔案，用來存放所有scss。

在helpers內的_variables.scss目前是空的，我們需要找到
node_modules > Bootstrap > variables
然後把variables內的資料複製並貼到helper內的variables。

流程四匯入必要資料到all.scss

如果使用Bootstrap客製化，有幾隻檔案必須要先匯入到all.scss

一樣把../node_modules/替換成~符號，這裡的variables改成使用我們自定的也就是helpers內的variables 。

流程五修改原本路徑

接著把一開始流程二內的SCSS路徑改成自己的路徑。

完成以後就可以修改variables內的變數了。
